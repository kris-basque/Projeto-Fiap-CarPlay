# CarPlay UI Concept — FIAP

A CarPlay interface concept developed for FIAP's Systems Analysis program. Focused on in-car UX design translated into front-end code.

---

## About

This project explores automotive UI design principles — how to present information clearly in a high-distraction environment. Built with plain HTML and CSS, prioritizing layout, hierarchy, and readability.

## Tech Stack

- HTML5
- CSS3
